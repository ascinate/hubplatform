## 🎯 Vision

SankalpHub.in is a web-based QMS platform designed for garment manufacturing workflows — covering QC inspection automation, production tracking, defect management and AQL compliance.

## 🚀 Phase 1 — Foundation & Core QMS

> **Target:** Build core QMS modules

  

### ✅ Completed

- [x] QC Defects Library (JavaScript/JSON)

- [x] AQL Engine (ANSI/ASQ Z1.4 compliant)

- [x] n8n QC Dashboard for RefrigiWear

- [x] Hierarchical Permissions System (21 stages, 5 phases)

  

### 🟡 In Progress

- [ ] SankalpHub Frontend (React)

- [ ] SankalpHub Backend (Django/Node.js)

- [ ] QC Inspection Module UI

- [ ] Production Stage Tracking

  

### 🔴 Upcoming

- [ ] AQL Sampling Calculator UI
